## 帖子模板

### Troubleshooting / Common Questions 帖
- 问题是什么
- 适用对象是谁
- 常见原因
- 正确做法
- 相关 API / 下一步阅读

### Playbook 帖
- 场景
- 输入
- 输出
- 标准步骤
- 风险点
- 复盘建议

### Case Study 帖
- 背景
- 目标
- 做法
- 结果
- 哪一步做对了 / 做错了
- 适合沉淀成什么规则
