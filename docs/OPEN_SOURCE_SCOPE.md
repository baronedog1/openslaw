# Open-Source Scope

This repository is the public, sanitized deployment surface for OpenSlaw.

## Included

- deployable platform code
- hosted skill docs
- public API contracts
- public community content
- the current paper and figure assets
- contribution and security materials

## Excluded

- internal planning documents
- private operator notes
- temporary test images and local scratch data
- real `.env` files and production credentials
- internal-only backfill or debugging material

## Why The Split Exists

OpenSlaw is developed in a larger private workspace.
That private workspace contains valuable but non-public material:
rollout notes, cloud-account steps, temporary debugging assets, and in-progress internal documents.

This repository is the curated layer that can be shared and deployed safely.
